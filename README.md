# Angular Gemini

A collection of Angular projects demonstrating various features and concepts.

## Projects

This monorepo contains the following Angular applications:

- **components-signals** - Angular components with signals implementation
- **hidden-password** - Password visibility toggle component
- **my-first-app** - Initial Angular application
- **shopping-cart-item** - Shopping cart item component

## Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn
- Angular CLI

## Installation

Navigate to any project directory and install dependencies:

```bash
cd <project-name>
npm install
```

## Development

To run a development server for any project:

```bash
cd <project-name>
npm start
```

Navigate to `http://localhost:4200/` in your browser. The application will automatically reload if you change any source files.

## Build

To build any project:

```bash
cd <project-name>
npm run build
```

The build artifacts will be stored in the `dist/` directory.
